01 / Course Description

An introduction to the principles of computer security: cryptography (symmetric, asymmetric, hashing), software security (memory-safety vulnerabilities and mitigations), access control, malware, and network attacks. Lecture plus hands-on labs.

02 / Learning Outcomes

By the end of the term, students will be able to:

  1. Explain the CIA triad and apply it to threat models.
  2. Distinguish symmetric, asymmetric, and hash primitives and select the right tool.
  3. Describe classic memory-safety vulnerabilities and OS/compiler mitigations.
  4. Compare discretionary and mandatory access-control models.
  5. Classify malware and reason about propagation, payload, and stealth.
